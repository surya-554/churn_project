🎯 Customer Churn Predictor
A simple Machine Learning web app built using Python and Streamlit to predict whether a customer is likely to leave a company (customer churn prediction).

🚀 Features


Predict customer churn risk


Simple and beginner-friendly UI


Built with Streamlit


Uses trained Machine Learning model


Instant prediction with probability score



🛠️ Technologies Used


Python


Streamlit


NumPy


Pickle


Machine Learning



📂 Project Structure
churn_project/│├── app.py├── churn_model.pkl├── WA_Fn-UseC_-Telco-Customer-Churn.csv└── README.md

▶️ How to Run the Project
1️⃣ Install Required Libraries
pip install streamlit numpy scikit-learn pandas

2️⃣ Open Project Folder
cd churn_project

3️⃣ Run Streamlit App
streamlit run app.py

🎮 How to Use


Select customer tenure


Enter monthly charges


Click Predict Now


View churn prediction result



📊 Prediction Output


✅ LOW RISK → Customer likely stays


⚠️ HIGH RISK → Customer may leave
